• En variabel består av ett namn och ett värde
  • Variabelns namn och värde paras ihop med ett likhetstecken (=)
x = 17

  Här paras namnet x ihop med värdet 17.

  • Du kan välja vilket namn som helst på din variabel och då är det bra att välja något som beskriver vad den ska spara. Till exempel kan vi vilja spara antalet passagerare:
antal_passagerare = 3

Variabler kan uppdateras

  • Du kan när som helst ändra värdet på en variabel.
x = 17
x = 3
x = -5

Efter att ha kört koden ovan kommer namnet x vara ihopkopplat med värdet -5 och kopplingen mellan de andra har försvunnit.

  • Ett till exempel:
antal_passagerare = 3
antal_passagerare = antal_passagerare + 1

Efter att ha kört koden ovan kommer variabeln med namnet antal_passagerare att ha värdet 4. På andra raden kommer datorn först leta i minnet efter en variabel med namn antal_passagerare och ta dess värde som är 3. Sedan kommer datorn addera 3 + 1 = 4. Sist kommer då värdet 4 att paras ihop med namnet antal_passagerar och variabeln kommer sparas i minnet.

Variabler kan innehålla olika sorters värden

  • Värdet på variabler kan vara av olika datatyper. Hittills har vi använt heltal men andra exempel är datatyper som strängar och flyttal. Mer om dessa kan du läsa under avsnittet om datatyper.
